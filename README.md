## Installation

1.) Clone this repository

2.) Locate to the project directory

3.) Install the packages that is needed for this boilerplate.
```bash
npm install
```
### Packages Included
- `express`
- `cors`
- `morgan`
- `nodemon`
- `dotenv`
- `body-parser`

4.) Start the server
```bash
npm run dev:start
```
or
```bash
npm start
```
and you're done!
